◆ ADLX_APP_GPU_DEPENDENCY

enum ADLX_APP_GPU_DEPENDENCY

Indicates the type of dependence of an application on the GPU on which it runs.

Enumerator
APP_GPU_UNKNOWN 

It is unknown if the application is bound to the GPU on which it runs. The application might behave unexpectedly if the GPU is powered off or disabled.

APP_GPU_BOUND 

The application is bound to the GPU on which it runs and will behave unexpectedly if the GPU is powered off or disabled.

APP_GPU_NOT_BOUND 

The application is not bound to this GPU on which it runs and will continue to run properly if the GPU is powered off or disabled.
