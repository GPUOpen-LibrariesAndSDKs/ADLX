Demonstrates how to obtain the Display FreeSync color accuracy when programming with ADLX and perform related operations.

Command Prompts

Command PromptDescription
1 Display FreeSync color accuracy support.
2 Get FreeSync color accuracy state.
3 Set current FreeSync color accuracy to disabled.
4 Set current FreeSync color accuracy to enabled.
M/m Display the command prompt menu.
Q/q Terminate the application.
