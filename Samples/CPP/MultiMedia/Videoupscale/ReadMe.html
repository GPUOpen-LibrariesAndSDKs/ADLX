Demonstrates how to access video upscale options, and perform related testing when programming with ADLX.

Command Prompts

Command PromptDescription
1 Display video upscale support
2 Display current video upscale state, current sharpness, and sharpness range
3 Enable video upscale and change minimum sharpness
4 Disable video upscale
M/m Display the main menu
Q/q Quit the application
