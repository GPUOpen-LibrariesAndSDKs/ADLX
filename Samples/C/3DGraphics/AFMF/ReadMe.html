Demonstrates how to access AMD Fluid Motion Frames options, and perform related testing when programming with ADLX.

Command Prompts

Command PromptDescription
1 Display AMD Fluid Motion Frames support
2 Display current AMD Fluid Motion Frames state
3 Enable AMD Fluid Motion Frames
4 Disable AMD Fluid Motion Frames
M/m Display the main menu
Q/q Quit the application
